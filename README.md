# Memory-Analysis
A social psychologist wanted to determine whether there are differences in memory for emotional and neutral events. The psychologist randomly assigned 20 participants to study a set of negative, positive, or neutral pictures. Each participant was instructed to study the set for 1 minute, and after a 5-minute delay, they were asked to recall as many pictures as possible.
